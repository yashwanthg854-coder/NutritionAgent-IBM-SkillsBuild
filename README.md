## Problem Statement
Good nutrition is the foundation of health, but individuals often struggle with diet planning, portion control, and tracking nutritional intake.

## Proposed Solution
A multi-agent AI-powered Nutrition Agent built using LangFlow, IBM watsonx.ai, and IBM Granite Models to deliver personalized, real-time dietary guidance using RAG.

## Technology Stack
- IBM Granite Models
- IBM watsonx.ai
- IBM watsonx Orchestrate
- RAG (Retrieval-Augmented Generation)
- Vector Database (FAISS/Chroma)
